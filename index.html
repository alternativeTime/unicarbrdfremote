Welcome to the UniCarbKB RDF Model

This page should be read in conjunction with the GlycoRDF project site and GlyTouCan. This is a working document that provides information on the RDF model with a focus on glycoprotein specific data stored in UniCarbKB.

Refer to glycan.owl (proposed update)

Example RDF Triples

Glycan Sequence

<http://rdf.unicarbkb.org/glycan/123>
    a   glycan:saccharide ;
    glycan:has_glycosequence
    <http://rdf.unicarbkb.org/glycan/123/glycoct> 
    <http://rdf.unicarbkb.org/glycan/123/uoxf> .

GlycoCT Structure Representation

<http://rdf.glycoinfo.org/glycan/123/glycoct>
    a   glycan:glycosequence ;
    glycan:in_carbohydrate_format    
    glycan:carbohydrate_format_glycoct ;
    glycan:has_sequence 
        “RES…”^^xsd:string .


#### Oxford Notation
<http://rdf.unicarbkb.org/glycan/123/uoxf>
    a glycan:glycosequence ;
    glycan:in_carbohydrate_format    
    glycan:carbohydrate_format_uoxf ;
    glycan:has_uoxf
        "A2G(4)1"^^xsd:string .

Protein Summary

@PREFIX uniprot: <http://purl.uniprot.org/core/> .

    glycan:has_attached_glycan
        <http://rdf.unicarbkb.org/structure/705>…

        <http://rdf.unicarbkb.org/glycoAASite/195_SER107> , 
        <http://rdf.unicarbkb.org/glycoAASite/195_SER99> ;

        "uniprot:P00740" .
